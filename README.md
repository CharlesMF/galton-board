# 高爾頓板模擬器 (Galton Board Simulator)

一個使用 Matter.js 物理引擎製作的互動式高爾頓板視覺化程式，用於展示常態分佈（Normal Distribution）的形成過程。

## 📊 專案簡介

高爾頓板（Galton Board）又稱為釘板或豆機，是由英國統計學家 Francis Galton 發明的機械裝置。球從頂部落下，經過多層釘子的隨機碰撞後，最終在底部形成近似常態分佈的鐘形曲線。

本專案使用 HTML5 和 JavaScript 實現互動式模擬，無需安裝任何軟體，直接在瀏覽器中執行。

## ✨ 功能特色

- 🎯 真實物理模擬：使用 Matter.js 引擎模擬球體運動
- 🎨 視覺化效果：多彩球體和流暢動畫
- 🎮 互動操作：可自由增加球數和重置
- 📈 常態分佈展示：觀察統計學原理的實際應用

## 🚀 快速開始

### 線上執行

直接在瀏覽器中打開 `galton_board.html` 即可。

### 本地執行

1. 下載專案檔案
2. 雙擊 `galton_board.html` 或用瀏覽器開啟
3. 點擊按鈕增加球數，觀察分佈形成

## 🎮 使用說明

- **+50 顆球**：增加 50 個球體
- **+200 顆球**：增加 200 個球體
- **重置**：清空所有球體，重新開始

## 🔧 技術參數

- **釘子排列**：12 層三角形排列
- **物理設定**：低彈性（0.1）、輕微摩擦（0.05）
- **畫布尺寸**：700 x 850 像素
- **重力加速度**：1 單位

## 📚 相關知識

### 常態分佈（Normal Distribution）

常態分佈是統計學中最重要的機率分佈之一，又稱高斯分佈。在自然界和社會現象中廣泛存在，如：
- 人類身高分佈
- 考試成績分佈
- 測量誤差分佈

### 中央極限定理

高爾頓板完美展示了中央極限定理：大量獨立隨機事件的總和趨近於常態分佈。
